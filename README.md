# Coralite-Mod
这是一个泰拉瑞亚mod。
This is a Terraria mod.

本mod目的是为泰拉瑞亚增添更多的趣味，包括但不限于新的物品，敌怪，BOSS，新系统等。

将会包含植物学系统，植物学内容如下，目前正在制作中
	-采集和种植更加多样化的植物，并培育它们
	-杂交与突变，获得稀有的植物
	-完成植物图鉴
	-炼药系统，更加强大的植物将会提供更加强大的药水
	-更多的武器与装备
	-植物装饰物

关于BOSS
    -血肉墙预计添加4个BOSS，包括
        -赤玉灵（已完成）
        -冰龙宝宝（已完成）
        -XXX（沙漠BOSS，规划中）
        -影子球（规划中）
    -血肉墙后的BOSS
        -正在规划中，暂时不会透露相关内容
